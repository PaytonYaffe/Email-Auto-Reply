# Email-Auto-Reply
AI-powered email auto-reply web app built with Django. Accepts incoming email messages, generates contextual replies via an AI API, and stores inquiry history in a database with a Django admin interface.

A full-stack Django web application that automatically generates replies to incoming email messages using an AI API. Users can submit an email inquiry through the web interface, receive an AI-generated response in real time, and review a persistent history of all past inquiries and replies. Built with Django, SQLite, and vanilla JavaScript, with a Django admin panel for database management.
Tech stack: Python, Django, JavaScript, HTML/CSS, SQLite
